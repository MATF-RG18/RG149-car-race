Kratko uputstvo:


Cilj igre je da se sakupi sto vise novcica, nakon odredjenog vremena brzina kretanja
lopte ce da se postepeno povecava. Ukoliko lopta udari u rozu prepreku, igra se zaustavlja.

Tasteri koje mozemo da koristimo su:

esc => za izlazak iz igrice.
a => za skretanje u levo.
d => za skretanje u desno.
r => za restartovanje (skor se resetuje na 0, kao i brzina lopte, kao i teren)
p => za pauziranje. Kada prvi put kliknemo dugme p pauzirace se igrica,
na drugi klik ce da se nastavi dalje.



Odradio sam:
1. postavio scenu
2. napravio teksture
3. namestio kameru da prati playera
4. namestio animaciju
5. generisao prepreke
6. lopta se rotira oko sebe, kako bi izgledalo realnije da se kotrlja
7. generisu se novcici
8. ispisuje se Score
